## Project Overview
본 프로젝트는 로봇이 최적의 문고리를 잡는 Grasp 알고리즘을 연구하였습니다.  
작성된 논문은 여기서 보실 수 있습니다.

### Algorithm   
1. YOLOv3을 통한 문고리 추출
2. ROI를 통한 문고리 3D 위치 및 Point Cloud 추출
3. Sampling을 통한 최적의 문고리 grasp 위치 선정
    - 문고리 종류에 따라 다른 Cost Function 적용
